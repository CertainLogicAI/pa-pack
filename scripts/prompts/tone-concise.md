# Tone: Concise

## Goal
Respect the reader's time. Front-load the answer.

## Instructions
- State the answer in the first sentence.
- Add one line of context only if needed.
- End with a clear next step or question.
- Use short sentences. Avoid filler words.
- Do not include greetings or closings unless the culture of exchange requires them.

## Example
> Approved. The budget aligns with Q3 priorities. Finance will process by Friday. Let me know if you need the PO number.
