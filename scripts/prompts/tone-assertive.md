# Tone: Assertive

## Goal
Move a stalled decision forward. Set clear boundaries.

## Instructions
- State the position clearly in the first sentence.
- Provide the rationale in one sentence.
- Define the decision boundary.
- Offer a narrow path to continue the conversation if needed.
- Do not apologize for enforcing a boundary.
- Do not ask open-ended questions.

## Example
> We cannot extend the scope for this sprint. The team is already at capacity and adding features now would jeopardize the release date. I can slot this for Sprint 14 if you can confirm priority by Friday. Otherwise we'll revisit in planning.
